Experience

Hardware Intern at Extreme Networks

I served in this position over the summer of 2019. Here were some of my responsibilities:
Created Excel Add-in and program that gathers data from testing equipment and inserts it into an Excel spreadsheet. Speeds up the test data gathering process and reduces human error.
Created a (Python) GUI program to connect via Telnet or Serial to a POE Loader, send commands, and monitor current states across specified ports. Created a distributable that ran the program without any extra download.

Created (Python) program to connect via Telnet to a switch, run a function from a provided C file, and save the output to a specified file (.csv, .txt, etc). This program has both a GUI and a console interface. Created distributable package that ran the program without any extra download.
Wrote multiple C functions that can be used on switches to configure, activate, and view information about ports.
Assisted with and learned about the preliminary steps to adding a Secure Boot system to Extreme switches. Tested writing to a crypto authentication chip via I2C from a Raspberry Pi.
Assisted in troubleshooting and repair of various Extreme switches, including installing and configuring operating systems, as well as writing programs to connect to the switch and read information over time.
